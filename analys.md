# Госсарий терминов
## Основные понятия
- **Дашборд (Dashboard)**: Информационная панель, объединяющая графики, метрики и виджеты для быстрого анализа данных.
- **KPI (Key Performance Indicator)**: Ключевой показатель эффективности, измеряющий успех достижения целей.
- **Виджет (Widget)**: Отдельный элемент дашборда, такой как график или счетчик.

## Типы дашбордов
- **Интерактивный дашборд**: Панель с элементами, реагирующими на действия пользователя, как фильтры или клики.
- **Оперативный дашборд**: Обновляется в реальном времени для текущего мониторинга процессов.
- **Стратегический дашборд**: Фокусируется на долгосрочных KPI и трендах компании.

## Элементы визуализации
- **Диаграмма (Chart)**: Графическое представление данных, включая круговые или линейные графики.
- **Тепловая карта (Heatmap)**: Визуализация распределения значений по цветовой шкале.
- **Прогресс-бар (Progress Bar)**: Индикатор заполнения цели в процентах.

## Обработка данных
- **Фильтрация (Filtering)**: Отбор данных по заданным критериям.
- **Агрегация (Aggregation)**: Сводка данных, например, сумма или среднее.
- **Тренд (Trend)**: Направление изменения показателя во времени.

## Дополнительные функции
- **Датасет (Dataset)**: Подготовленный набор данных для дашборда.
- **Модель данных (Data Model)**: Структура связей между таблицами и метриками.
- **Отклонение (Variance)**: Разница между плановым и фактическим значением.
- **Скользящее среднее (Moving Average)**: Среднее значение за скользящий период для сглаживания.
- **Self-service**: Дашборд, настраиваемый пользователями без IT-специалистов.
- **BI-платформа**: Система вроде Power BI или Tableau для создания дашбордов.
- **Группировка (Grouping)**: Объединение данных по категориям.
- **Сортировка (Sorting)**: Упорядочивание данных по значениям.
