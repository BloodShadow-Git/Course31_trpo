Данное ПО предназначено для визуализации данных.
В базовый набор модулей включены:
- Excel - модуль загрузки данных из файлов *.xlsx

При запуске программа автоматически ищет файлы "отчётов" для их загруки и визуализации. Какие модули следует запускать можно настроить.

Используемые библиотеки:
- MAUI 10
- R3
- ObservableColections
- R3.MAUI
- R3.ObservableCollections
- Newtonsoft.Json
Программа свободна для коммерческого использования. Любые идеи для улучшения и исправления приветствуются :)
